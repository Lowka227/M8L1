# Телеграм-бот для технической поддержки

Этот бот помогает пользователям получать ответы на часто задаваемые вопросы и направлять запросы специалистам.

---

## Инструменты для реализации

- **SQL** для хранения запросов
- **telebot** для баз данных
- **googletrans** для перевода сообщений
- **speech_recognition** для перевода голосовых сообщений в текст

---

##  Проект состоит из:

| Файл | Описание |
|------|-----------|
| `bot.py` | Основной файл бота |
| `logic.py` | Логика бота |
|`config.py`| Настройки бота |
| `support.db` | База данных для запросов |

---

##  Как использовать

Для работы с программой убедитесь, что у вас установлены все библиотеки.

---

### Установка

1. Клонируйте репозиторий на свой локальный компьютер:
    ```bash
    git clone путь_к_репозиторию
    ```
2. Вставьте токен бота в файл `config.py`
3. Запустите бота:
   ```bash
   python bot.py
